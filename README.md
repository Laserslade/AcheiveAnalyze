# AcheiveAnalyze
This project is an HTML application where educators can input in CSVs of students grades along side with assignemnts via an API of an LLM no matter the formattting of the CSV the AI parses only the nessecary information and feeds it into a linear regression model to predict student grades on future assignments. This tool also uses monte carlo simulations for enhanced predicitons and AI powered feedback individually to students.
